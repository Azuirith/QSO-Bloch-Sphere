# QSO-Bloch-Sphere

This project is the hardware/software implementation of the Purdue Quantum Student Organization (QSO) Bloch Sphere. 
Here we integrate Qiskit with Arduino using Python to create an interface between quantum logic gates and motors that rotate
a physical representation of a qubit in a quantum system.
